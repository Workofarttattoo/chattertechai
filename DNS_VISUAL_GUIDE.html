Configure chattertechai.com with Google Workspace DNS

1

Access Google Domains Settings

Choose one of these options:

Option A: Google Workspace Admin

  1. Go to admin.google.com
  2. Click Domains
  3. Click Manage domains
  4. Click chattertechai.com
  5. Click DNS

Option B: Google Domains Direct

  1. Go to domains.google.com
  2. Click chattertechai.com
  3. Click DNS in left menu
Quick Link:
Open Google Domains ‚Üí
2

Add 4 A Records (GitHub Pages IPs)

Click "Manage custom records" or "Add record", then add these 4 records:

Host name Type TTL Data
@ (or leave blank) A 1 hour 185.199.108.153
@ (or leave blank) A 1 hour 185.199.109.153
@ (or leave blank) A 1 hour 185.199.110.153
@ (or leave blank) A 1 hour 185.199.111.153
‚ö†Ô∏è Important: Use @ symbol (or leave blank) for "Host name", NOT "chattertechai.com"

What you're entering (for each record):

  • Host name: Type @ (this means root domain)
  • Type: Select A from dropdown
  • TTL: Leave as default (1 hour)
  • Data/Points to: Copy/paste one IP address
  • Click "Add" or "Save"
  • Repeat 4 times (one for each IP)
3

Add CNAME Record for www

Add one more record for the www subdomain:

Host name Type TTL Data
www CNAME 1 hour workofarttattoo.github.io

What you're entering:

  • Host name: Type www
  • Type: Select CNAME from dropdown
  • TTL: Leave as default (1 hour)
  • Data/Points to: Type workofarttattoo.github.io
  • Click "Add" or "Save"
‚ö†Ô∏è Don't delete MX records! Keep all existing MX records for your email to work. Only ADD the records above.
4

Save & Wait for DNS Propagation

After adding all 5 records (4 A + 1 CNAME):

  1. Click "Save" at the bottom of the page
  2. DNS propagation takes 10-60 minutes (usually ~15 minutes)
  3. Don't panic if it doesn't work immediately!

Check DNS Propagation Status:

Check if DNS has propagated ‚Üí whatsmydns.net

You should see all 4 GitHub Pages IP addresses when DNS is ready:
185.199.108.153, 185.199.109.153, 185.199.110.153, 185.199.111.153

5

Configure Custom Domain in GitHub

Once DNS has propagated (wait 15+ minutes):

  1. Go to GitHub Pages settings
  2. Under "Custom domain", enter: chattertechai.com
  3. Click "Save"
  4. Wait 1-2 minutes for DNS check
  5. Check the "Enforce HTTPS" box once available
